<a name="install"></a>
<h1>Installation</h1>

##1. Clone the project via git clone command

##2. Install npm packages
```bash
npm install
```

##3. Run the project

Run the project in development mode
```bash
npm run dev
```

Build the project for production
```bash
npm run build
```

Run the latest production build
```bash
npm run peview
```
